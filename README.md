To run, you will first need to make sure the "devtools" and "learnr" packages are installed. 
Then install the package "PositPrimers", using the following code: devtools::install_github("liannesiegel/PositPrimers”).
To run a tutorial, run: learnr::run_tutorial("[Insert name of tutorial]", "PositPrimers")
