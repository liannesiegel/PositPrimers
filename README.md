To run, you will first need to make sure the "devtools" and "learnr" packages are installed. 
Then install the package "PositPrimers", using the following code: devtools::install_github("liannesiegel/PositPrimers”).
To run a tutorial, run: learnr::run_tutorial("[Insert name of tutorial]", "PositPrimers")

The package contains the following tutorials: 

Basics:
"Programming-Basics" 
"Visualization-Basics" 

ggplot2: 
"ggplot2-Exploratory-Data-Analysis"
"ggplot2-Bar-Charts"
"ggplot2-Histograms"
"ggplot2-Boxplots"
"ggplot2-Scatterplots"
"ggplot2-Line-Graphs"
"ggplot2-Overplotting"
"ggplot2-Customize" 

dplyr: 
"dplyr-tibbles" 
"dplyr-isolating"
"dplyr-deriving"
